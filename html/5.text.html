macam macam font

  1. font family
    adalah untuk mengatur font kita ada yang arial, san-serif dll
  2. font-size
    adalah untuk mengatur ukuran font kita bisa ligter,blod,bolder dll
  3. font-weight
    adalah untuk mengatur tebal kecilnya huruf klo font-size hanya mengubah ukuranya saja
  4. font-decoration
    adalah untuk mengatur garis bawah sebuah font mau ditiadakan atau diadakan
  5. font-style
    untuk mengubah font menjadi miring

macam text

  1. text shadow
    adalah untuk mengatur bayangan yang terdapat pada huruf di web kita
  2. word-spacing
    untuk mengatur jarak antara kata perkata
  3. letter-spacing
    untuk mengatur jarak perhuruf

tautan

  1. tautan:hover
    untuk mengubah font warna yang ada di link ini berfungsi ketika kursor kita di diamkan di link tersebut akan berubah warna
  2. tautan:active
    untuk mengubah font link ketika kita tekan
  3. tautan:visited
    untuk mengubah warna font ketika kita sudah pernah mengujungi nlink
