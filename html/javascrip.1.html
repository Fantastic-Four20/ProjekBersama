variable
Variabel pada Java digunakan untuk menyimpan nilai tertentu di dalam memori. Sebuah variabel didefinisikan dengan menggunakan sebuah nama dan tipe data yang spesifik. Tipe data ini menentukan jenis nilai yang dapat disimpan di dalam variabel tersebut. Sebagai contoh, variabel yang bertipe data integer hanya dapat menyimpan nilai bilangan bulat. Dalam Java, kita dapat mendeklarasikan variabel dengan cara berikut:
let
ok kita pengenalan dengan variabel pertama kita yaitu yang bernama let. let adalah sebuah variabel yang berfungsi klo di ibaratkan adalah sebuah memori yang menyimpan sebuah file langsung aja kita ke contohnya tampa banyak bacot

var
ok ini adalah contoh variable kita adalah var ya klao di lihat lihat sama saja fungsinya untuk menjadi memori lu klo bedain ajalah sendiri apa bedanya pusing gw
