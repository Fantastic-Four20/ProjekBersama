border

apa itu border? border adalah untuk mengatur sebuah garis dibawah sebuah kontent di css kita dan banyak lagi kegunaanya yang tidak kita ketahui dan dibawah ini kita kan menjelaskan macam macamnya

  1. border sytle
    2. untuk mengatur sytle border kita dan kita bisa pindahkan border kita terletak dimana
  2. border color
    3. untuk mengatur warnanya yang terdapat di border kita
  3. border size
    4. untuk mengatur ke tebalan / tipis sebuah border
ini border sytle ini border color ini border size
